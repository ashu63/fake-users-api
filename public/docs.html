How to use it

Fake Users API can be used with any type of project that needs todos, comments and users in JSON format. you can use examples below to check how Fake Users API works and feel free to enjoy it in your awesome projects!

Guide

You can copy paste the code in your browser console to quickly test Fake Users API.

Creating a User

fetch('https://jsonplaceholder.typicode.com/posts', { method: 'POST', body: JSON.stringify({ name: 'foo', email: 'mhavvock1e@usnews.com', phone: 137-251-1505, address: 4 Burrows Circle, age: 51, }), headers: { 'Content-type': 'application/json; charset=UTF-8', }, }) .then((response) => response.json()) .then((json) => console.log(json));
